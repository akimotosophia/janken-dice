# じゃんけんダイス 放置×ガチャゲーム

サイコロの6面にグー・チョキ・パーが書いてあり、戦略的にカスタマイズして戦う放置型ガチャゲームです。

## ゲームルール

### 基本ルール
- サイコロの6面にグー・チョキ・パーが配置されている
- 毎回サイコロを振って、相手のサイコロと勝負
- じゃんけんのルールで勝てばポイント獲得

### 戦略要素
- **面のカスタマイズ**: サイコロの面の構成を自由に組み替えられる
- **相手の分析**: 相手の面の構成も見れるので、最適な組み合わせを選べる
- **特殊な面**: ガチャで特殊な面を獲得できる
  - 例: ダブルグー = 勝った時にポイント2倍
  - その他、様々な効果を持つ面が登場

### ガチャシステム
- ポイントを使ってガチャを回す
- 新しい特殊な面を獲得
- レアな面ほど強力な効果

## 技術スタック

- **HTML5**: ゲームの構造
- **CSS3**: アニメーション、グロー効果、レスポンシブデザイン
- **Canvas API**: パーティクルエフェクト(コイン、光の粒子など)
- **Vanilla JavaScript**: ゲームロジック、アニメーション制御

## 実装予定の演出

- サイコロの回転アニメーション
- 勝利時のグロー/光るエフェクト
- コインや光の粒子のパーティクルエフェクト
- スムーズな画面遷移

## 開発

ローカルで開発する場合は、HTMLファイルをブラウザで直接開くか、ローカルサーバーを起動してください。

```bash
# 例: Pythonのシンプルサーバー
python -m http.server 8000
```
