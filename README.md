# ゲームハブ - 放置×ガチャゲーム集

複数の放置型ガチャゲームを楽しめるゲームハブです。

## プロジェクト構成

```
janken-dice-1/
├── index.html          # ゲーム一覧ページ（トップページ）
├── common.css          # 共通スタイル
├── common.js           # 共通JavaScript（背景アニメーション）
├── netlify.toml        # Netlify設定
├── README.md           # このファイル
└── games/              # ゲームディレクトリ
    └── janken-dice/    # じゃんけんダイス
        ├── index.html
        ├── styles.css
        └── script.js
```

## 収録ゲーム

### 1. じゃんけんダイス

サイコロの6面にグー・チョキ・パーが書いてあり、戦略的にカスタマイズして戦う放置型ガチャゲーム。

#### ゲームルール
- サイコロの6面にグー・チョキ・パーが配置されている
- 毎回サイコロを振って、相手のサイコロと勝負
- じゃんけんのルールで勝てばポイント獲得

#### 戦略要素
- **面のカスタマイズ**: サイコロの面の構成を自由に組み替えられる
- **相手の分析**: 相手の面の構成も見れるので、最適な組み合わせを選べる
- **特殊な面**: ガチャで特殊な面を獲得できる
  - 例: ダブルグー = 勝った時にポイント2倍
  - その他、様々な効果を持つ面が登場

#### ガチャシステム
- ポイントを使ってガチャを回す
- 新しい特殊な面を獲得
- レアな面ほど強力な効果

## 技術スタック

- **HTML5**: ゲームの構造
- **CSS3**: アニメーション、グロー効果、レスポンシブデザイン
- **Canvas API**: パーティクルエフェクト(コイン、光の粒子、背景アニメーション)
- **Vanilla JavaScript**: ゲームロジック、アニメーション制御

## 実装済みの機能

### 共通機能
- 背景のパーティクルアニメーション
- レスポンシブデザイン
- ゲーム一覧カード

### じゃんけんダイス
- サイコロの回転アニメーション
- 勝利時のグロー/光るエフェクト
- コインや光の粒子のパーティクルエフェクト
- 放置ポイント獲得システム
- ガチャシステム（4段階のレアリティ）
- サイコロカスタマイズ機能

## 開発

ローカルで開発する場合は、HTMLファイルをブラウザで直接開くか、ローカルサーバーを起動してください。

```bash
# 例: Pythonのシンプルサーバー
python -m http.server 8000

# アクセス
# トップページ: http://localhost:8000/
# じゃんけんダイス: http://localhost:8000/games/janken-dice/
```

## 新しいゲームの追加方法

1. `games/` ディレクトリ内に新しいゲーム用フォルダを作成
2. ゲームのHTML、CSS、JavaScriptファイルを配置
3. `index.html` のゲーム一覧に新しいカードを追加

## デプロイ
